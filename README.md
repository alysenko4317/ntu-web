# Основи веб-технологій — супутні матеріали курсу

Репозиторій містить навчальні матеріали до дисципліни **«Основи веб-технологій»**: демонстраційний код і приклади для лекцій, а також завдання та еталонні рішення для лабораторних робіт.

Матеріали розділено на дві основні частини:

| Розділ | Каталог | Призначення |
|--------|---------|-------------|
| Лекції | [`Lections/`](Lections/) | Приклади коду, демо-проєкти та супровідні файли до лекційного матеріалу |
| Лабораторні роботи | [`Labs/`](Labs/) | Методичні вказівки (`.docx`) та вихідний код для виконання ЛР |

## Зміст курсу (огляд)

Курс охоплює повний цикл розробки веб-застосунків — від клієнтських технологій і мережевих основ до серверної логіки, фреймворків і контейнеризації:

1. **Клієнт–серверна взаємодія** — HTTP, веб-клієнти та прості сервери (Java, Python).
2. **Фронтенд** — HTML, CSS, JavaScript (DOM, події, AJAX, Web Workers).
3. **Серверна розробка** — WSGI, PHP (сесії, авторизація, MVC).
4. **Фреймворк Django** — вступ, моделі, автентифікація.
5. **Інфраструктура** — Docker, nginx, розгортання застосунків.

Лабораторні роботи 1–5 утворюють наскрізний ланцюжок: від статичного сайту до PHP-застосунку з авторизацією та MVC-архітектурою. **Кожен студент розвиває власний проєкт** — предметну галузь і структуру сторінок обирають у ЛР1 і дотримуються її в усіх наступних роботах.

> **Про еталонний код у репозиторії.** Каталоги `LR3_library_static_site`, `LR4_library_php_auth` та (за наявності) `LR5_library_php_auth_mvc` містять демонстраційну реалізацію на темі **«електронна бібліотека»**. Це референс для викладача та орієнтир для студентів, а не обов'язкова тема для всіх. Технічні вимоги (HTML/CSS/JS, PHP, сесії, MVC) однакові; змінюються лише назви сторінок, поля форм і текстовий контент відповідно до обраного домену.

## Структура репозиторію

```text
.
├── Lections/          # Матеріали до лекцій (див. Lections/README.md)
│   ├── L2 - WebClients/
│   ├── L3_4 - HTML/
│   ├── L5_6 - CSS/
│   ├── L7_8_9 - JS/
│   ├── L13 - WSGI/
│   ├── L14 - Django Intro/
│   ├── L15 - Django Models/
│   ├── L16 - Auth/
│   └── L17 - Docker/
│
└── Labs/              # Лабораторні роботи (див. Labs/README.md)
    ├── LR1_html_css_basics/
    ├── LR2_networks_basics/
    ├── LR3_library_static_site/
    ├── LR4_library_php_auth/
    └── LR5_library_php_auth_mvc/
```

> **Примітка.** Нумерація лекцій у репозиторії відповідає навчальному плану курсу; окремі лекції можуть не мати власної папки з кодом, якщо матеріал носить переважно теоретичний характер.

## Як користуватися

### Лекції

Перейдіть до [`Lections/README.md`](Lections/README.md) — там зібрано покажчик усіх тем із коротким описом і посиланнями на відповідні каталоги. Більшість фронтенд-прикладів можна відкрити напряму у браузері; серверні демо (Python, Django, Docker) потребують відповідного середовища — деталі вказано в README кожної лекції.

### Лабораторні роботи

Перейдіть до [`Labs/README.md`](Labs/README.md). Для кожної ЛР наведено методичний документ (`.docx`) і, за наявності, каталог `src/` з еталонним кодом. У ЛР1 студент обирає власну тематику сайту; еталони ЛР3–ЛР5 ілюструють один із можливих варіантів (бібліотека). Окремі README є в папках ЛР3 і ЛР4.

## Необхідне програмне забезпечення

Залежно від теми, студентам може знадобитися:

- **Браузер** з інструментами розробника (Chrome, Firefox тощо)
- **Apache HTTP Server** + PHP (для лабораторних з PHP)
- **Python 3** + pip (Flask, Django, Waitress)
- **Java JDK** + Tomcat 9 (для Java-демо на L2)
- **Docker** (для L17 та HTML-демо з Docker)
- **IDE**: PyCharm, IntelliJ IDEA, VS Code

## Ліцензія та використання

Матеріали призначені для навчального процесу в рамках курсу «Основи веб-технологій». При використанні поза навчальним контекстом бажано посилатися на джерело.
