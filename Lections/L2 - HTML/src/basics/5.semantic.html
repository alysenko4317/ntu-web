Що таке HTML?

HTML (HyperText Markup Language) — мова розмітки, яка описує структуру вебсторінки. Браузер інтерпретує HTML-код і відображає його у вигляді візуального документа.

Основні принципи

  • HTML задає структуру, а не зовнішній вигляд.
  • CSS відповідає за оформлення.
  • JavaScript забезпечує інтерактивність.

Семантичні теги HTML5

Семантичні теги (<header>, <nav>, <main>, <article>, <section>, <aside>, <footer>) роблять структуру сторінки зрозумілою для браузерів, пошукових систем і допоміжних технологій.
