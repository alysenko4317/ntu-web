# Лекція 4. Вступ до JavaScript

## Тема
JavaScript як мова програмування веб-браузера: базовий синтаксис, типи даних, функції, масиви, об’єкти та взаємодія з DOM.

## Мета
Сформувати початкове уявлення про роль JavaScript у веб-розробці та навчити використовувати базові конструкції мови для керування поведінкою веб-сторінки.

## Очікувані результати
Після лекції студент повинен уміти:
- пояснити роль JavaScript поруч з HTML і CSS;
- відрізнити ECMAScript від API середовища виконання;
- підключити сценарій до HTML;
- використовувати `let`, `const` і розуміти особливості `var`;
- працювати з основними типами даних;
- пояснювати динамічну й слабку типізацію;
- використовувати умови, цикли та функції;
- працювати з масивами й об’єктами;
- пояснювати різницю між `==` і `===`;
- знаходити та змінювати елементи DOM;
- реєструвати обробники подій;
- користуватися Console і debugger.
