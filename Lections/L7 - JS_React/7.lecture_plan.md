# Лекція 7. Знайомство з React і створення першого проєкту

## Тема лекції

**Основи React: компонентна модель, JSX, властивості, стан і створення першого застосунку за допомогою Vite.**

## Мета лекції

Сформувати розуміння компонентного й декларативного підходів до побудови вебінтерфейсів, ознайомити зі структурою React-проєкту та навчити створювати функціональні компоненти, передавати їм дані й керувати локальним станом.

## Очікувані результати навчання

Після лекції студент повинен уміти:

- пояснити, яку проблему розв’язує React;
- порівняти ручне оновлення DOM із декларативним підходом;
- пояснити роль Node.js, npm і Vite;
- створити та запустити новий React-проєкт;
- описати основні файли й каталоги проєкту;
- пояснити призначення JSX;
- створити функціональний компонент;
- експортувати й імпортувати компоненти;
- вкладати компоненти один в один;
- передавати дані через `props`;
- будувати список компонентів через `map`;
- пояснити призначення `key`;
- пояснити загальну ідею Virtual DOM;
- реєструвати обробники подій;
- використовувати `useState`;
- пояснити повторний рендеринг;
- користуватися React Developer Tools.

## Рекомендовані демонстрації

Код і посилання: [`README.md`](README.md), каталог [`src/`](src/).

1. Порівняти [`src/pure_js/index.htm`](src/pure_js/index.htm) і [`src/react/index.htm`](src/react/index.htm).
2. Перевірити версії Node.js і npm.
3. Відкрити готовий Vite-проєкт [`src/sample0/react/`](src/sample0/react/) (`npm install`, `npm run dev`).
4. Розібрати `package.json` і структуру каталогів (пояснити, чому `node_modules` не в git).
5. Показати `index.html`, `main.tsx` та `App.tsx`.
6. Пройти кроки [`src/sample0/steps/`](src/sample0/steps/) 0→7 (JSX → компонент → props → map → лайки).
7. Винести компонент у окремий каталог (крок 6).
8. Використати компонент кілька разів / `map` + `key`.
9. Видалити `key` і показати попередження.
10. Показати Increment/Decrement без стану ([`sample1/steps/4.app-incr-decr`](src/sample1/steps/4.app-incr-decr/)).
11. Замінити оновлення звичайної змінної на `useState` (крок 7 / `Video`).
12. Показати незалежний стан кількох карток.
13. Продемонструвати старе значення після setter-виклику (`console.log` одразу після `setLikesCount`).
14. Розглянути `props` на кількох екземплярах [`src/sample2/react-app/`](src/sample2/react-app/).
15. Увімкнути підсвічування рендерингів у React Developer Tools.
