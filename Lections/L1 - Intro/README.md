# Лекція 1. Вступ до вебтехнологій

## Тема

**Вступ до вебтехнологій та принципів роботи веб-застосунків.**

## Основний напрямок

Лекція формує загальне уявлення про розвиток вебу, будову сучасних веб-застосунків і принципи клієнт-серверної взаємодії. Розглядається обмін даними між браузером і сервером, призначення основних веб-технологій та способи адресації ресурсів у мережі.

## Розглянуті питання

- поняття веб-технологій і веб-застосунку;
- клієнтська та серверна частини веб-системи;
- статичні й динамічні веб-сторінки;
- роль HTML, CSS і JavaScript;
- серверна логіка та взаємодія з базами даних;
- розвиток вебу від статичних документів до інтерактивних застосунків;
- принципи роботи HTTP та HTTPS;
- структура URL;
- доменні імена, IP-адреси, DNS і мережеві порти;
- абсолютні та відносні адреси ресурсів;
- підключення стилів, сценаріїв, зображень та інших файлів.
