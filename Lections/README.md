# Матеріали до лекцій

Покажчик демонстраційного коду та супровідних файлів до лекцій курсу **«Основи веб-технологій»**.

У каталозі кожної лекції типово є:

| Файл | Призначення |
|------|-------------|
| `README.md` | Опис тематики лекції та пов’язаних матеріалів (приклади, зв’язок з ЛР) |
| `N.lecture_plan.md` / `N.lection_plan.md` | План лекції |
| `N.lecture_notes.md` / `N.lection_notes.md` | Конспект лекції |
| `src/` | Демонстраційні приклади (якщо є) |

| Лекція | Тема | Каталог |
|--------|------|---------|
| L1 | Вступ до вебтехнологій | [`L1 - Intro/`](L1%20-%20Intro/) |
| L2 | HTML, структура веб-сторінки, Docker, сучасні веб-API | [`L2 - HTML/`](L2%20-%20HTML/) |
| L3 | CSS, Bootstrap, адаптивна верстка | [`L3 - CSS/`](L3%20-%20CSS/) |
