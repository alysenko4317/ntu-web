to open and run java version oh simplest http-server you can use IntelliJ Pro (you can use trial or educational license)
   - open foledr 'java\hello_servlet' as a project
   - you will require to create run configuration with Tomcat 9, so it will also required to install Tomcat 9

for python version you can use PyCharm Community Edition
   - open foledr 'python\simple_http_server' as a project
   - you will require to create two run configurations - one per each .py file
   - it may require you to install additional packages like "flask";
        it is recomended to use virtual python environment for that project;
        you can create the one from IDE or command line
