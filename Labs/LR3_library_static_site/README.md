# Library Base — статичний навчальний сайт (еталон ЛР3)

Демонстраційна статична версія сайту **електронної бібліотеки**. Показує очікувану технічну реалізацію ЛР3 (HTML, CSS, JavaScript), але **не є обов'язковою темою** для студентів.

У **ЛР1** кожен студент обирає власну предметну галузь і створює сайт під неї. У ЛР3 потрібно розвинути саме **свій** проєкт — додати JavaScript, форми з клієнтською перевіркою та інтерактивність. Еталон у `src/` можна використовувати як орієнтир за структурою коду, адаптуючи сторінки, поля форм і тексти під обраний домен.

## Сторінки (у демо-еталоні)

- `index.html` — головна сторінка;
- `catalog.html` — каталог книг із клієнтським пошуком;
- `about.html` — опис бібліотеки;
- `details.html` — опис розвитку проєкту;
- `contacts.html` — контактна форма з клієнтською перевіркою;
- `login.html` — статична форма входу;
- `register.html` — статична форма реєстрації;
- `cabinet.html` — макет особистого кабінету.

## Каталоги

- `css/style.css` — стилі сайту;
- `js/main.js` — клієнтська інтерактивність і перевірка форм.

## Як використовувати

Сайт можна відкрити напряму з файлової системи або розгорнути через Apache HTTP Server.

У **ЛР4** статичну версію **власного** сайту перетворюють на PHP-застосунок:

- перейменувати потрібні `.html` файли у `.php`;
- додати `session_start()`;
- реалізувати серверну обробку форм;
- додати реєстрацію, авторизацію і захищений кабінет.

Демонстраційний PHP-еталон для бібліотеки — у [`LR4_library_php_auth`](../LR4_library_php_auth/).

### Docker (для зручності)

Проєкт містить `Dockerfile` та `run.bat` для швидкого запуску через Docker:

```
run.bat
```

Сайт буде доступний за адресою: **http://localhost:8083**
