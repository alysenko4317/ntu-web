Для чого ця версія сайту?

Вона імітує результат попередніх лабораторних робіт: студенти вже мають готовий статичний сайт, який можна розгортати через Apache HTTP Server.

Що буде далі?

На наступній лабораторній роботі сторінки можна перетворити на PHP-сторінки, підключити серверну обробку форм, сесії та захищений кабінет користувача.
