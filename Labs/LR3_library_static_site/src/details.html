Як розвивається сайт

Ця сторінка пояснює, як статичний сайт може поступово перетворюватися на серверний веб-застосунок.

1

HTML і CSS

Створюється структура сторінок, навігація, каталог, форми та оформлення.

2

JavaScript

Додаються пошук, адаптивне меню, перевірка форм і невелика інтерактивність.

3

PHP

Форми починають надсилати дані на сервер, з’являється авторизація та сесії.

4

База даних або Java

Проєкт можна розширити збереженням даних у БД або перенести логіку на Java Servlet/JSP.
