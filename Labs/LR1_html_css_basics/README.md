# Library Base — статичний HTML + CSS сайт (еталон ЛР1)

Демонстраційний статичний сайт **електронної бібліотеки** для лабораторної роботи №1. Показує очікувану технічну реалізацію: багатосторінковий сайт на HTML і CSS, розгорнутий на Apache HTTP Server.

> **Тема «електронна бібліотека» — демонстраційна.** У ЛР1 кожен студент самостійно обирає предметну галузь і створює сайт під обрану тематику. Еталон можна використовувати як орієнтир за структурою.

## Сторінки

- `index.html` — головна сторінка з описом проєкту та популярними книгами;
- `catalog.html` — статичний каталог книг;
- `about.html` — опис бібліотеки та ролі користувачів;
- `contacts.html` — контактна форма (статична, без обробки);
- `login.html` — форма входу (статична);
- `register.html` — форма реєстрації (статична);
- `cabinet.html` — макет особистого кабінету.

## Каталоги

- `css/style.css` — стилі сайту (адаптивна верстка, CSS-змінні).

## Як розгорнути

### Варіант 1: Apache у WSL (рекомендовано для ЛР1)

1. Встановити Apache у WSL:
   ```bash
   sudo apt update && sudo apt install apache2
   ```

2. Скопіювати файли сайту:
   ```bash
   sudo cp -r src/* /var/www/html/
   ```

3. Запустити Apache:
   ```bash
   sudo service apache2 start
   ```

4. Відкрити у браузері: http://localhost/

### Варіант 2: Docker (для зручності)

```
run.bat
```

Сайт буде доступний за адресою: **http://localhost:8081**
