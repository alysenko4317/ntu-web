Електронна бібліотека на HTML та CSS

Це базова статична версія сайту, створена у лабораторній роботі №1 засобами HTML і CSS. Сайт демонструє багатосторінкову структуру, адаптивну верстку та форми без серверної обробки.

Для чого ця версія сайту?

Вона демонструє результат лабораторної роботи №1: студенти створюють багатосторінковий статичний сайт на HTML і CSS, який можна розгортати через Apache HTTP Server.

Що буде далі?

У ЛР3 до сайту буде додано JavaScript: клієнтський пошук у каталозі, перевірку форм та адаптивне меню. У ЛР4 сторінки перетворяться на PHP-сторінки з серверною обробкою форм, сесіями та захищеним кабінетом користувача.
